## 🚀 環境構築

お使いのOSに合わせた手順で、開発環境のセットアップを行ってください。
このスクリプトは、ローカルでのHTTPS通信を可能にするために `mkcert` を使用し、信頼されたローカル認証局（CA）のインストールと、`localhost`用の証明書生成を自動で行います。

### Windows の場合

1.  **PowerShell** を開きます。
2.  プロジェクトのディレクトリに移動します。
3.  以下のコマンドを実行して、セットアップスクリプトを起動します。

    ```powershell
    ./setup.ps1
    ```

    もしスクリプトの実行がセキュリティポリシーによってブロックされる場合は、先に以下のコマンドを実行してください。
    `Set-ExecutionPolicy RemoteSigned -Scope Process -Force`

    このスクリプトは、パッケージマネージャの `winget` を利用して、Node.js, Python, mkcert が見つからない場合に自動でインストールを試みます。その後、プロジェクトに必要なライブラリをすべてインストールします。

### Linux / macOS の場合

ターミナルを開き、以下のコマンドを実行してください。

```bash
./setup.sh
```

このスクリプトは、必要なツールがインストールされているかを確認し、プロジェクトに必要なライブラリをすべてインストールします。`mkcert` が見つからない場合は、Homebrew (macOS) や apt/dnf (Linux) を使ったインストール方法を案内します。

## ▶️ アプリケーションの起動

環境構築が完了したら、以下のコマンドでメインのWebアプリケーションを起動できます。

```bash
node webnew/server.js
```

アプリケーションは `https://localhost:3000` でアクセス可能になります。
