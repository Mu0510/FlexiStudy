# プロジェクトの目的

このプロジェクトは、ユーザーの学習をサポートする「勉強・受験万能秘書」としてのCLIツールと、学習ログを視覚化・管理するためのWeb学習ダッシュボードを提供します。

## 主要な目標

*   **学習サポート:** ユーザーの学習状況を記録し、目標達成を支援します。
*   **学習ログの視覚化:** Webダッシュボードを通じて、学習の進捗や傾向を直感的に把握できるようにします。
*   **AIによる学習サポートの高度化:** 将来的には、AIを活用した学習パターンの分析、弱点分析、課題推薦など、より高度な学習支援機能を目指します。
*   **FlexiJournal構想:** 手書きとデジタルの融合による次世代手帳アプリケーションの開発を通じて、ユーザー中心の革新的なUXを提供します。
