# コードスタイルと規約

## TypeScript / Next.js
*   **言語:** TypeScript
*   **コンパイラオプション:** `tsconfig.json`に基づき、`strict`モードが有効。
*   **リンティング/フォーマット:** `next lint`スクリプトを使用。ESLintとPrettierがNext.jsに統合されているため、これらのツールによる自動チェックとフォーマットが適用されます。
*   **CSS:** Tailwind CSSを使用。Tailwind CSSのユーティリティファーストの原則と規約に従ってスタイリングを行います。

## Python
*   **言語:** Python 3
*   **スタイル:** 明示的なスタイルガイド（例: Black, Flake8）は設定されていません。既存の`manage_log.py`などのファイルから、以下の慣習を推測します。
    *   **インデント:** スペース4つ。
    *   **命名規則:** 変数名、関数名は`snake_case`。クラス名は`CamelCase`。
    *   **コメント:** 必要に応じてコードの意図や複雑なロジックを説明するコメントを追加します。
    *   **型ヒント:** 必要に応じて型ヒントを使用します。
